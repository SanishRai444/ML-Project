About This Project

This project is an AI-powered pollution detection system that analyzes satellite data to monitor air quality across different regions. It uses heatmap-like images generated from Earth observation satellites to classify whether an area is polluted or not

.

What Does It Do?

Monitors Air Pollution: The system collects weekly pollution data from satellite sources such as Sentinel-5P.

Processes Multiple Pollutants: It considers gases like NO₂, SO₂, CO, O₃, and Aerosol Optical Depth(AOD) to assess pollution levels.

Generates Composite Images: The collected data is averaged over a week to produce meaningful and stable pollution heatmaps.

Classifies Pollution Levels: A Convolutional Neural Network (CNN) model is trained to classify areas as either polluted or not polluted based on these weekly images.

User Interaction: Users can check the pollution status of any location by selecting it, and the system will analyze and show the latest results.

How Does It Work?

Satellite Data Collection: Pollutant bands are extracted using the Google Earth Engine API.

Image Processing: A weekly mean image is generated for each pollutant.

AI Classification: These processed images are passed through a trained CNN model that predicts whether the area is polluted.

Result Visualization: The system displays the classification result and can optionally show the pollution distribution in image form.
